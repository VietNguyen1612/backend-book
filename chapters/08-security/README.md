# Chapter 8: Security

Security is a discipline applied at every layer of a backend system, from application code to infrastructure. This chapter covers the essential practices for building secure applications and protecting your infrastructure.

## Table of Contents

1. [8.1 Application Security](application-security.md) -- OWASP Top 10 essentials (injection, broken authentication, XSS, CSRF, broken access control, security misconfiguration, insecure deserialization), input validation, security headers, CORS, file upload security, and rate limiting.

2. [8.2 Infrastructure Security](infrastructure-security.md) -- Secrets management (HashiCorp Vault, AWS Secrets Manager, environment variables), encryption at rest and in transit, GDPR compliance, dependency scanning, penetration testing, and zero-trust architecture.
